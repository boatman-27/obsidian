# React Components

This folder contains the following components, each linked to its detailed documentation:

- [[404]]: A component designed to handle 404 errors, typically shown when a page is not found. Useful for improving user experience by providing a friendly error page.
- [[Button]]: A reusable and customisable button component. It supports various styles, sizes, and onClick events to suit different use cases.
- [[DarkModeToggle]]: A component that provides a toggle switch for enabling or disabling dark mode. Includes smooth transitions and local state management.
- [[Helpers]]: A collection of utility functions and helper methods used throughout the application. These functions simplify tasks like formatting, calculations, or data manipulation.
- [[NavBar]] The navigation bar component, including links to different pages and a responsive design. Integrates seamlessly with routing and styling frameworks.
- [[Toaster]]: A notification component for displaying success, error, or informational messages. Features customisable duration and styles.

## Notes

- Each component includes its own Markdown file with detailed documentation, examples, and usage instructions.
- When adding a new component, create a corresponding Markdown file and link it here.
- Ensure each component adheres to consistent coding and styling standards.
- Explore possibilities to optimize and extend components for better performance and scalability.
