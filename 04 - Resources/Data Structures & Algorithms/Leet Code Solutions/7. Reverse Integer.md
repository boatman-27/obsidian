# [Problem 7] Reverse Integer

## Problem Statement
we are given a 32 bit integer and we reverse it...

If reversing `x` causes the value to go outside the signed 32-bit integer range `[-231, 231 - 1]`, then return `0`.

---
## Examples 
### **Example 1:**

**Input:** x = 123
**Output:** 321

### **Example 2:**

**Input:** x = -123
**Output:** -321

### **Example 3:**

**Input:** x = 120
**Output:** 21
