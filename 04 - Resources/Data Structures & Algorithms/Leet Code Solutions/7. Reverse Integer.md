# [Problem 7] Reverse Integer

## Problem Statement
we are given a 32 bit integer and we reverse it...

If reversing `x` causes the value to go outside the signed 32-bit integer range `[-231, 231 - 1]`, then return `0`.

---
## Examples 
### **Example 1:**

**Input:** x = 123
**Output:** 321

### **Example 2:**

**Input:** x = -123
**Output:** -321

### **Example 3:**

**Input:** x = 120
**Output:** 21

---
## Approaches 

i was thinking of using the old getting last digit and using modulo to remove it, then multiply result * 10 and last digit

---
## Code (in Go)
```go
package main

import (
	"fmt"
)

const (
	INT_MAX = 1<<31 - 1 //  2147483647
	INT_MIN = -1 << 31  // -2147483648
)

func reverse(x int) int {
	var res int
	for x != 0 {
		lastDig := x % 10
		x /= 10
		res = res*10 + lastDig
	}

	if res > INT_MAX || res < INT_MIN {
		return 0
	}
	return res
}

func main() {
	fmt.Println(reverse(1534236469))
}go
```
